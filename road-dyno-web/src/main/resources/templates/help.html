Road dyno Help

What is road dyno?

With Road Dyno, you can create accurate dyno graphs based on data taken by ECU logs. In order to achieve that, you need to log a few key parameters while making wide open throttle runs in one gear.

What should be logged

In order for log file to be used in road dyno, they need contain data from the follwing parameters:
  • RPM: If your data logger has more than one entry for RPM such as "Engine RPM" or "Shaft Speed", pick the entry labeled simply "RPM" when possible.
  • Engine Runtime: Always make sure that engine runtime is included in your logs.
